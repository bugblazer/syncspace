# SyncSpace

> Open-source, self-hosted team communication platform built for teams.

## Why SyncSpace?

SyncSpace aims to provide a modern, self-hosted alternative to paid communication platforms that teams can deploy on their own infrastructure while keeping complete ownership of their data.

---

## Tech Stack

### Backend

* Node.js
* Express.js
* PostgreSQL
* Prisma ORM
* Socket.io

### Frontend

* React
* Tailwind CSS
* Zustand

### DevOps

* Docker
* Docker Compose
